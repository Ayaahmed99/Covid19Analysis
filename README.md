# COVID-19 Data Analysis in SQL Server
This project analyzes global COVID-19 data using SQL Server. 
The analysis focuses on infection rates, death percentages, and vaccination progress across countries and continents.

The goal is to transform raw data into meaningful insights using advanced SQL techniques.
## Business Questions Answered

- Which countries had the highest infection rate?
- What is the global death percentage?
- Which continent was most affected?
- How did vaccination progress over time?

## SQL Skills Demonstrated

- Data cleaning and filtering
- Aggregate functions (SUM, MAX)
- Window functions (SUM OVER PARTITION BY)
- Common Table Expressions (CTEs)
- Temporary tables
- View creation
- Error handling with NULLIF()

## Dataset

The dataset contains global COVID-19 statistics including:
- Total and new cases
- Total and new deaths
- Population data
- Vaccination records
